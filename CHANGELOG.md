# Changelog

Все значимые изменения в проекте фиксируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
проект использует [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.7.0] - 2025-11-07

### Добавлено

- Поддержка конфигураций Frontmatter.

## [0.6.1] - 2025-11-06

### Изменено

- Обработан случай, если файл для преобразования не существует по указанному пути.
- Рефакторинг процесса конвертации.

## [0.6.0] - 2025-10-29

### Добавлено

- Поддержка таблиц с автонумерацией.

## Изменено

## [0.5.0] - 2025-10-29

### Добавлено

- Поддержка листингов кода с автонумерацией.

## [0.4.0] - 2025-09-17
